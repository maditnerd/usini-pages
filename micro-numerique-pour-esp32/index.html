    Micro numérique pour ESP32

    J'avais acheté un micro numérique (en I2S) mais je n'avais pas trouvé le temps de le tester depuis, sauf erreur il faut au moins un esp (je teste sur un esp32) pour qu'il fonctionne.

    J'ai galéré à trouver un code pour arduino en voici un qui marche https://revspace.nl/EspAudioSensor
    https://github.com/bertrik/NoiseLevel

    Cette exemple gère plusieurs bandes de fréquences et est omnidirectionnel (la portée est bien meilleur que les micros analogiques de base).
